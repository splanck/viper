Release notes and announcements for the Zanna platform.

v0.2.7 June 2026

Graphics3D Stability, ZannaIDE Workbench & Debugger, Fail-Closed Hardening

Graphics3D stops flickering and gains spot-light shadows, a deterministic multi-threaded software rasterizer, and node animation over a rebuilt glTF/FBX importer; ZannaIDE becomes a real workbench with a VM debugger, git source control, multi-root workspaces, and an integrated terminal; a new agent-facing CLI gives editors and scripts machine-readable diagnostics; and a runtime hardening backbone makes recoverable traps and asset loaders fail closed while the tree-walking VM and both bytecode engines produce identical results.
v0.2.6 June 2026

Game3D Engine, ZannaIDE Editor, Pointer-Safety Gate, Deep Hardening

A code-first Zanna.Game3D engine over the 3D runtime, a real ZannaIDE editor with build/run and live language services, a pointer-safety gate that removes raw Ptr from both frontends, Zia language-stability work (defer, structured try/catch/finally, Result[T]), a breaking standard-library namespace de-clutter, multi-backend 3D graphics and Physics3D hardening with a from-scratch glTF/FBX/OBJ asset pipeline, native cross-platform toolchain installers, and a hardening sweep across memory, threads, crypto, networking, the native toolchain, and packaging.
v0.2.5 May 2026

Runtime Hardening, HTTP/2, Localization, Compiler Throughput

Hardening pass across every runtime subsystem, ~40 new 2D graphics classes, full 3D asset pipeline with glTF/FBX skeletons and skinning, TLS-backed HTTPS/WSS servers, from-scratch HTTP/2, native RSA + X.509 validator, Zanna.Localization.* (11-class namespace, zero deps), structured diagnostics with caret underlines, parallel IL optimizer and codegen, zero-copy codegen pipeline, text-mode baseball franchise simulator demo.
v0.2.4 April 2026

Game Engine, Native Linker, 3D Backends, Toolchain Packaging

Full PE/COFF + ELF dynamic linking, Metal & D3D11 GPU backends feature-complete, PBR materials, Physics3D + Collider3D, MJPEG/Theora video, JPEG/PNG/GIF/OGG/MP3 codecs, ZPAK asset embedding, 10 new game engine systems, 17K-line XENOSCAPE sidescroller, 3D bowling demo, native macOS menu bar, GUI hover events & overlay render pass, Zia variadic params + lambdas + type aliases, `entity`/`value` renamed to `class`/`struct`.
v0.2.3 March 2026

Architectural Hardening

PAC/BTI security, ICE diagnostics, shared peephole abstractions, test consolidation.
v0.2.2 February 2026

Enum Support & Weakness Remediation

Full enum support in both Zia and BASIC, register coalescer, network test portability.
v0.2.1 January 2026

Performance & Codegen Improvements

Immediate operand optimization, MIR-level DCE, call-aware register allocation, strength reduction.
