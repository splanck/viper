Web Server

Zia 1,350 LOC

A multi-threaded HTTP server with routing, request logging, and static file serving. Demonstrates Viper's networking and threading capabilities.

Screenshot coming soon

Features

  • HTTP request parsing and routing
  • Static file serving with content-type detection
  • JSON API endpoints
  • Thread pool for concurrent request handling
  • Request logging system
  • Dynamic route handler registration

Build & Run

# Build all demos
./scripts/build_demos.sh

# Run this demo
viper run examples/apps/webserver/
