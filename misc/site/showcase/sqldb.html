ViperSQL

Zia 78,300 LOC

A PostgreSQL-compatible database engine written entirely in Zia. Features MVCC transactions, write-ahead logging, B-tree indexes, and PG wire protocol support for standard SQL clients. Ships with a `vsql` interactive client.

Screenshot coming soon

Features

  • PostgreSQL-compatible wire protocol
  • Multi-Version Concurrency Control (MVCC)
  • Write-Ahead Logging (WAL) for crash durability
  • B-tree indexes for efficient query execution
  • Full SQL language support (DDL, DML, window functions with PARTITION BY, set operations, JOIN with IS NULL, ALTER TABLE)
  • Session management and stored procedures
  • Interactive `vsql` REPL client
  • System views and metadata tables
  • CSV import/export with validation

Build & Run

# Build all demos
./scripts/build_demos.sh

# Run the database server
viper run examples/apps/vipersql/

# Or use the interactive client
viper run examples/apps/vipersql/vsql.zia
