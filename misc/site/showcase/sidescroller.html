XENOSCAPE

Zia 17,400 LOC

A Metroid-style sidescroller exploring an alien planet. Ten interconnected levels across six biomes, 25+ enemies, mid-bosses, and a final boss arena. Built end-to-end in Zia using the Viper game engine, runtime asset embedding (VPA), and the same native ARM64/x86-64 codegen used to build Viper itself. The largest game demo in the Viper showcase.

Screenshot coming soon

Features

  • Tilemap-based world with physics, gravity, and AABB collision
  • Camera with smooth follow + deadzone tracking
  • 25+ enemy types using composable Behavior presets (Patrol, Chase, Shoot, EdgeReverse, SineFloat)
  • Boss fights with multi-phase encounter scripts
  • Player abilities: double jump, dash, projectile attacks, health pickups
  • JSON level loading via `LevelDocument` (objects, spawn points, theme metadata)
  • Save/load via SaveData (versioned JSON, GC-managed strings)
  • Six biomes with unique tilesets, parallax backgrounds, and particle effects
  • HUD via `Viper.Game.UI` (bars, labels, achievement popups)
  • Audio: ambient music streaming + spatial sound effects
  • Native cross-platform: macOS / Linux / Windows (incl. ARM64 native smoke)

Build & Run

# Build all demos
./scripts/build_demos.sh

# Run this demo
viper run examples/games/xenoscape/
