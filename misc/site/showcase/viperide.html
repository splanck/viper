ViperIDE

Zia 2,792 LOC

An integrated development environment written in Zia with professional IDE features including IntelliSense, tabbed editing, and build system integration.

Screenshot coming soon

Features

  • Tabbed document editor with multiple open files
  • 6-phase IntelliSense code completion
  • Syntax highlighting for Zia and BASIC
  • Find and replace with regex support
  • Integrated build system
  • Project tree navigation
  • Font zoom and theme toggling
  • Settings persistence
  • F11 fullscreen mode

Build & Run

# Build all demos
./scripts/build_demos.sh

# Run this demo
viper run examples/apps/viperide/
