ViperIDE

Zia 3,300 LOC

An integrated development environment written in Zia using the Viper GUI toolkit. Live diagnostics, hover tooltips, IntelliSense, go-to-definition, project search, and a code editor with multi-cursor editing.

Screenshot coming soon

Features

  • Tabbed document editor with multiple open files
  • Live diagnostics powered by `Viper.Zia.Completion.Check`
  • Hover tooltips with type and signature info (`Viper.Zia.Completion.Hover`)
  • IntelliSense code completion (6-phase pipeline: Sema → Engine → C bridge → runtime.def → UI)
  • Symbol outline via `Viper.Zia.Completion.Symbols`
  • Project search across the workspace
  • Multi-cursor editing in the code editor widget
  • Syntax highlighting for Zia and BASIC
  • Find and replace with regex support
  • Integrated build system
  • Project tree navigation
  • Font zoom and dark/light theme toggling
  • Settings persistence (HiDPI scale, theme, font size, last project)
  • F11 fullscreen mode
  • Native menu bar mirroring on macOS (Cmd+N, Cmd+S, Cmd+Q)

Build & Run

# Build all demos
./scripts/build_demos.sh

# Run this demo
viper run examples/apps/viperide/
