Chess

Zia 4,000+ LOC

A polished Zia chess game with alpha-beta AI, save slots, built-in puzzles, analysis mode, themed Canvas UI, clocks, audio, and FEN/PGN support.

Screenshot coming soon

Features

  • Complete chess rule enforcement (castling, en passant, promotion)
  • Alpha-beta search with iterative deepening and named engine personalities
  • Quiescence search for sound evaluation
  • Transposition table for move caching
  • Save/resume, settings, pause menu, and persistent stats/achievements
  • Casual, blitz, rapid, and classical clocks
  • Board themes, piece themes, accessibility toggles, and assist overlays
  • Canvas-rendered board with drag-and-drop, replay undo/redo, and move history
  • Built-in lessons, analysis board, FEN export/import helpers, and PGN export

Build & Run

# Build all demos
./scripts/build_demos.sh

# Run this demo
viper run examples/games/chess/
