3D Bowling

Zia 4,600 LOC

A complete 3D bowling game built on Viper's from-scratch Graphics3D and Physics3D runtimes. Demonstrates Metal / Direct3D 11 / OpenGL / software backend parity, rigid-body pin collision with continuous collision detection, ball spin and hook, four camera modes, and the full ten-frame scoring including strikes, spares, and the tenth-frame fill ball.

Screenshot coming soon

Features

  • Physics3D rigid bodies with quaternion orientation, angular velocity, sleep system, and CCD substep sweeps
  • Collider3D shapes: box (lane), capsule (pin), sphere (ball), with proper restitution and friction
  • Ball spin and hook via angular impulse + lateral force during release
  • Material3D with Blinn-Phong lighting; all four 3D backends supported (Metal on macOS, D3D11 on Windows, OpenGL on Linux, software fallback)
  • Four camera modes: chase, behind-pins, top-down, free-fly
  • Full bowling scoring: strikes, spares, double/triple strikes, tenth-frame fill ball
  • Animated pin reset and lane sweep between frames
  • HUD with frame-by-frame score display and current roll indicator
  • Native cross-platform: macOS / Linux / Windows

Build & Run

# Build all demos
./scripts/build_demos.sh

# Run this demo
viper run examples/games/3dbowling/
