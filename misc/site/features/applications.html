Real Applications

Viper isn't a "hello world" compiler. The examples directory contains production-quality applications and games totaling over 100,000 lines of code — all written in Viper's own languages and compiling to native binaries.

What This Proves

A 73,000-line SQL database with MVCC and a wire protocol isn't a toy example. A chess engine with transposition tables and a drag-drop GUI isn't a demo. These are real applications that exercise every layer of the platform:

  • Language features — Classes, inheritance, generics, pattern matching, modules
  • Runtime capabilities — Graphics, networking, threading, crypto, file I/O
  • Compiler quality — These programs run correctly at scale, handling edge cases across 73K+ lines
  • Performance — Games run at interactive frame rates; the database handles concurrent transactions
