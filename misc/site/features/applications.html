What This Proves

A 78,000-line PostgreSQL-compatible database with MVCC and a wire protocol isn't a toy example. A 17K-line Metroid-style sidescroller, a 3D bowling game with full rigid-body physics, an IDE with IntelliSense and live diagnostics — none of these are demos. They are real applications that exercise every layer of the platform:

  • Language features — Classes, structs, inheritance, generics, lambdas, variadic params, pattern matching, modules, typed exception handling
  • Runtime capabilities — 2D + 3D graphics, audio (WAV/OGG/MP3), video, networking (HTTP server, WebSocket, TLS), threading, crypto, GUI widgets, asset embedding
  • Compiler quality — These programs run correctly at scale, handling edge cases across tens of thousands of lines
  • Performance — Games run at interactive frame rates; the database handles concurrent transactions
