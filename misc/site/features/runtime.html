Full-Stack Runtime

Viper's runtime library is portable C implementing 300 classes across 22 modules. It rivals the standard libraries of established platforms like Python, Java, or C# — but every line is hand-written with zero external dependencies.

Platform-specific backends use only OS-native APIs: Cocoa / Metal on macOS, X11 / OpenGL on Linux, Win32 / D3D11 on Windows. A pure-software 3D rasterizer is always available as a fallback.

GUI Toolkit (46 classes)

Cross-platform desktop widgets for building IDEs and tools. Includes buttons, labels, text input, sliders, modals, dropdowns, menubars, tab bars, tree views, code editor with multi-cursor, scroll views with clipping, floating panels, video widgets, and more. Recent additions: full hover events (MOUSE_ENTER/MOUSE_LEAVE), justify-content layout (CENTER/SPACE_BETWEEN/SPACE_AROUND/SPACE_EVENLY), per-app theme ownership, and platform text input (`VGFX_EVENT_TEXT_INPUT`) for non-QWERTY keyboards.
