# Appendix A: Zia Reference

The complete, maintained Zia language reference lives outside this book:

- [Zia Language Reference](../../languages/zia-reference.md) — syntax, semantics, types, and grammar
- [Zia Grammar](../../languages/zia-grammar.md) — formal grammar notes

Use those documents, plus `viper check`, as the source of truth for syntax and
supported features. Keeping one canonical reference means this book never
drifts out of date as the language evolves.
