Full-Stack Runtime

Viper's runtime library is 146,872 lines of portable C implementing 229 classes across 12+ modules. It rivals the standard libraries of established platforms like Python, Java, or C# — but every line is hand-written with zero external dependencies.

Platform-specific backends use only OS-native APIs: Cocoa on macOS, X11 on Linux, Win32 on Windows.
