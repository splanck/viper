Paint

Zia 2,410 LOC

A drawing application inspired by MS Paint, built with Viper's Canvas API. Features 8 drawing tools, a color picker, and undo/redo support.

Screenshot coming soon

Features

  • 8 drawing tools: pencil, brush, eraser, line, rectangle, ellipse, fill, eyedropper
  • Custom color palette with HSL/RGB support
  • Adjustable brush size and opacity
  • Undo/redo capability
  • UI sliders and buttons
  • Real-time canvas rendering

Build & Run

# Build all demos
./scripts/build_demos.sh

# Run this demo
viper run examples/apps/paint/
