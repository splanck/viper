Nova Run

Zia 6,592 LOC

A 2D platformer with jetpack physics, parallax scrolling, particle effects, 5 themed levels, and boss fights. The largest game demo in the Viper showcase.

Screenshot coming soon

Features

  • Physics engine with gravity, velocity, and collision detection
  • Camera system that follows the player
  • Multiple enemy types with AI behavior
  • Parallax scrolling backgrounds
  • Particle effects (dust, explosions)
  • Collectible pickups (coins, health, power-ups)
  • Sound system integration
  • HUD with score, health, and lives
  • 5 themed levels with tile-based maps
  • Menu system with scene transitions
  • Boss fights

Build & Run

# Build all demos
./scripts/build_demos.sh

# Run this demo
viper run examples/games/sidescroller/
