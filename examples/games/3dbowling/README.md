# Neon Lanes Championship

Neon Lanes is the release-quality evolution of Viper's 3D Bowling showcase. It
combines regulation ten-pin scoring and physics-driven pin collisions with an
accessible timing release, tactical equipment, persistent progression, and
local arcade modes.

The completed 25-part implementation and corrected game defects are mapped in
[`UPGRADE_AUDIT.md`](UPGRADE_AUDIT.md).

## Modes

- **Quick Game** — a complete single-player ten-frame game.
- **Local Match** — two to four named hot-seat bowlers with individual colors
  and equipment choices.
- **Championship** — a three-rival tour through Riley Chen, Mateo Cruz, and
  Nova King. Every AI delivery uses the same physics, scoring, and camera path
  as the player.
- **Challenges** — Pocket Master, Spare School, and Clutch Tenth events with
  persistent bronze, silver, and gold medals.
- **Practice Lab** — free bowling plus repeatable 7–10, 10-pin, and 3–6–10
  spare drills.
- **Cosmic Party** — short Strike Rush, Low Ball, and accuracy-scored Target
  Bowling sessions with a dedicated neon lighting treatment.

## Controls

| Action | Keyboard | Gamepad |
|---|---|---|
| Aim / menu | A/D or arrows | D-pad |
| Shape hook | Q/E | Bumpers |
| Charge and release | Space | A |
| Previous / next ball | 1 / 3 | Stick buttons |
| Camera | C | Y |
| Restart | R | X |
| Pause / back | P or Escape | Start / B |

Hold the throw control to build power. The cyan timing marker moves across the
release strip; letting go near its green center produces the most accurate
launch. Standard and Full assists recover progressively more timing error,
while Pro retains nearly all of it.

## Release content

The demo owns its title key art and maple lane texture under `assets/`; the
texture also grounds the live oil-map preview. Missing assets fall back to the
live procedural venue and procedural board materials.
Music, ambience, and effects are generated by Viper's dependency-free audio
synth so the game remains completely self-contained.

Run from the repository root with:

```sh
viper run examples/games/3dbowling
```

The project manifest includes the `assets/` directory in macOS, Linux, Windows,
and portable packages, and uses the original 512-pixel bowling emblem under
`packaging/` as its application icon.

Engine/runtime behavior discovered while building the release upgrade is
tracked with minimal reproductions in
[`known_viper_issues/README.md`](known_viper_issues/README.md). Per the current
development constraint, those issues are documented here without modifying or
rebuilding Viper itself.
