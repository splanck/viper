Viper Web Server

A multithreaded HTTP server written in Zia

Welcome to the Viper Web Server demo. This server is written entirely in Zia, the primary frontend language for the Viper compiler toolchain.

Features

  • Multithreaded request handling via thread pool
  • Static file serving with MIME type detection
  • Path traversal protection
  • Configurable logging
  • Clean HTTP/1.1 implementation

Technology Stack

  • Language: Zia
  • Compiler: Viper
  • Runtime: Viper Runtime (networking, threading, file I/O)
